# Avatars Directory

This directory contains your AI avatars. Each avatar has its own subdirectory with:

- `config.yaml` - Avatar configuration
- `persona.md` - Avatar personality and instructions

## Creating an Avatar

Create a new directory with your avatar's ID and add the required files:

```
avatars/
  my-avatar/
    config.yaml
    persona.md
```

See the documentation for config.yaml schema and examples.
