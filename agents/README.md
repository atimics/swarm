# Agents Directory

This directory contains your AI agents. Each agent has its own subdirectory with:

- `config.yaml` - Agent configuration
- `persona.md` - Agent personality and instructions

## Creating an Agent

Create a new directory with your agent's ID and add the required files:

```
agents/
  my-agent/
    config.yaml
    persona.md
```

See the documentation for config.yaml schema and examples.
