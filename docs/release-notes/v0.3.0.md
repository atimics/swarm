# 🚀 RATi Avatar System v0.3.0 — Smarter Memory, Stronger Platforms

## 🧠 Semantic Memory Recall
Avatars now use **embedding-based semantic search** for memory retrieval. Instead of just recency, the system surfaces the most relevant memories by meaning. Long-running avatars should see noticeably better recall.

## 🔗 Wallet Linking
Link multiple wallets to your account **directly from chat**. Just ask "link a wallet" — connect, sign a message (no gas), done. All linked wallets visible in your account.

## 🖼️ Media URL Fix
Images, stickers, and audio now use **CDN URLs** instead of raw S3 links. Fixes broken thumbnails and inconsistent media loading.

## 🛡️ Discord Hardening
- ⏱️ **Rate limiting** — messages auto-queued to avoid API throttling
- 🔄 **Smart reconnect** — exponential backoff instead of hammering retry
- ✅ **Startup validation** — missing intents detected and reported clearly

## 🐦 X/Twitter Reliability

- 📊 Clear diagnostics when features are blocked by API tier
