# RATi Avatar System v0.13.1 - Gallery Recovery, Discord Hardening, Faster CI

## Highlights

This release is focused on reliability, not new surface area. It closes the
remaining gallery retrieval bug, hardens the Discord gateway reconnect path,
and trims CI wall time by removing duplicate work from the test job.

## Gallery Reliability

- Gallery list and search now paginate across the full result set instead of
  stopping at a shallow slice of recent rows.
- Type-filtered gallery queries now surface older matching items that were
  previously hidden behind newer non-matching media.
- The runtime gallery path now stays aligned with the normalized gallery ID
  contract introduced in `v0.13.0`.

## Discord Gateway Stability

- Gateway connections now enforce heartbeat ACK timeouts so stalled sessions
  reconnect instead of hanging silently.
- Heartbeat intervals are validated and jittered to reduce synchronized
  reconnect behavior.
- Resume failures now clear invalid session state instead of repeatedly trying
  to recover from a broken gateway session.
- Reconnect handling is tuned to reduce abnormal-closure churn seen in the
  shared Discord gateway worker.

## CI and Delivery

- The test job no longer pays for as much duplicate workspace build overhead,
  reducing end-to-end CI wall time for routine PR validation.
